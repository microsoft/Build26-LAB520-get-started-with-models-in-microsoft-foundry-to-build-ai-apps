# Lab Modules

Step-by-step instructions for each phase of the lab. You play the role of **Serena**, a developer at **Zava** (a global home-improvement retailer), building a product review moderation system.

| Phase | Module | Duration | Description |
|-------|--------|----------|-------------|
| 0 | [Setup](../setup/SETUP.md) | 10 min | Environment setup and authentication |
| 1 | [Discover Models](lab1-discover-models.md) | 10 min | Navigate the Foundry portal and explore the model catalog |
| 2 | [Project Setup](lab2-project-setup.md) | 10 min | Create a Foundry project and provision resources with azd |
| 3 | [Connect & Infer](lab3-connect-and-infer.md) | 12 min | SDK setup, authenticate, and send your first inference |
| 4 | [Review Moderation](lab4-comment-moderation.md) | 20 min | Build Zava's product review moderation pipeline |
| 5 | [Model Comparison](lab5-model-comparison.md) | 15 min | *(Self-paced extension -- not included in 75-min session)* Compare outputs across models |
| 6 | [Deploy Agent](lab6-deploy-agent.md) | 15 min | Package moderation as a hosted agent |
| 7 | [Summary](lab7-summary.md) | 5 min | Summary, Q&A, and next steps |
| - | [Cleanup](../cleanup/CLEANUP.md) | 5 min | Remove Azure resources |
