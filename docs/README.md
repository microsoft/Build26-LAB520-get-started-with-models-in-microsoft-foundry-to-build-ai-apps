# Lab Modules

Step-by-step instructions for each phase of the lab.

| Phase | Module | Duration | Description |
|-------|--------|----------|-------------|
| 0 | [Setup](../setup/SETUP.md) | 10 min | Environment setup and authentication |
| 1 | [Discover Models](lab1-discover-models.md) | 10 min | Navigate the Foundry portal and explore the model catalog |
| 2 | [Project Setup](lab2-project-setup.md) | 15 min | Create a Foundry project and provision resources with azd |
| 3 | [Connect & Infer](lab3-connect-and-infer.md) | 15 min | SDK setup, authenticate, and send your first inference |
| 4 | [Comment Moderation](lab4-comment-moderation.md) | 20 min | Build a working moderation pipeline |
| 5 | [Model Comparison](lab5-model-comparison.md) | 15 min | *(Optional extension — requires a second model deployment)* Compare outputs across models |
| 6 | [Deploy Agent](lab6-deploy-agent.md) | 20 min | Package moderation as a hosted agent |
| 7 | [Summary](lab7-summary.md) | 10 min | Workshop summary and next steps |
| - | [Cleanup](../cleanup/CLEANUP.md) | 5 min | Remove Azure resources |
